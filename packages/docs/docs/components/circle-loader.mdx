# Circle Loader

### Import

```jsx
import { CircleLoader } from "@sparrowengg/twigs-react";
```

### Usage 
```jsx live
<Box css={{ background:'$secondary100', padding:'$6' }}>
  <CircleLoader size="xl" />
</Box>
```

### Props
| Property       | Description                                | Type                    | Default |
|----------------|--------------------------------------------|-------------------------|---------|
| size           | size of the loader                         | `sm` `md` `lg` `xl` `2xl` `3xl`   | md     |
