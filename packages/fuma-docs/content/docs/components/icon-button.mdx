---
title: IconButton
description: A specialized button component designed to display icons with various sizes, colors, variants, and states. Perfect for action buttons, toolbars, and interface controls that require icon-based interactions.
---

## Import

```jsx
import { IconButton } from "@sparrowengg/twigs-react";
```

## Usage

```jsx
const IconButtonExample = () => {
  return (
    <IconButton
      icon={<AttachmentIcon />}
      aria-label="Edit"
      size="md"
      color="primary"
      variant="solid"
    />
  );
};
```
